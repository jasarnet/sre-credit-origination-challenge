# Estrategia de Confiabilidad

La confiabilidad se trata como un requerimiento funcional, no como un objetivo
aspiracional.

La estrategia se basa en:
- Medición objetiva de la experiencia del usuario
- Definición clara de fallas aceptables
- Degradación controlada y recuperación rápida

El sistema puede fallar, pero nunca debe hacerlo de forma silenciosa.
