# Contexto del Servicio

## Contexto de Negocio

La API de originación de créditos es un servicio crítico para el negocio.
Cualquier degradación o indisponibilidad impacta directamente en ingresos y en
la experiencia del cliente.

Actualmente, la observabilidad existe de forma básica, pero no hay una
estrategia formal de confiabilidad que permita detección temprana y una
respuesta efectiva a incidentes.
